## 🚀 How to Build APK

### Using Android Studio (Recommended)
1. Download Android Studio: https://developer.android.com/studio
2. Open Android Studio
3. Click "Open" and select this folder
4. Wait for Gradle sync (may take a few minutes first time)
5. Click **Build > Build Bundle(s) / APK(s) > Build APK(s)**
6. APK will be in: `app/build/outputs/apk/debug/app-debug.apk`

### For Release APK (Signed)
1. In Android Studio: **Build > Generate Signed Bundle / APK**
2. Select APK
3. Create or use existing keystore
4. Select release build type
5. APK will be ready for Play Store
